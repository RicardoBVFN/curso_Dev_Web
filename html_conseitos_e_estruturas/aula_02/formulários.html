Formulários servem para realizar a coleta de dados para um posterior armazenamento ou processamento pelo servidor do site. São delimitados pela tag <form>.

Existem ainda diversos tipos de imputs que podem ser ralisados em um formulário, tais como:

  • <input>: comenado responsável por criar um espaço para captação dos dados de entrada (números, textos, etc).
  • <label>: define um rótulo para os elementos do seu formulário.
  • <select>: cria uma lista suspensa de elementos pre definidos que o usuário deve escolher.
  • <textarea>: define um espaço para entrada de um texto com múltiplos caracteres.
  • <button>: cria um botão clicável que pode ser integrado com outros recursos.
  • <fieldset>: agrupa dados relacionados dentro de um formulário.
  • <legend>: define um texto como legenda de algum elemento "fieldset".
  • <datalist>: define uma série de opções pré definidas de input para o usuário escolher.
  • <output>: representa o output do servidor de um processamento de dados.
  • <option>: define uma opção em uma lista suspensa.
  • <optgroup>: define um grupo de opções em uma lista suspensa.
