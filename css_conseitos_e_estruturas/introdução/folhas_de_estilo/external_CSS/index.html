CSS externo

diferentemente das demais técnicas de aplicação de folhas de estilo, o CSS externo funciona a partir do vínculo entre um arquivo html e um outro arquivo apenas para o CSS, por meio de um link estabelecido no header do arquivo html.

este tipo de estilização, devido ao seu maior grau de organização, é o mais adotado no meio proficional, portanto, é considerado mais formal do que os demais.
