Tanto divs quanto spans servem para separar determinado bloco de conteúdo do resto do código, para uma estilisação mais eficiente

sendo as divs, delimitadas pela tag <div>, um elemento de bloco, o qual por padrão ocupa todo o espaço disponível na página

ja o span, delimitado pela tag <span>, é um elemento inline, ocupando, por padrão, apenas o espaço nescessário
