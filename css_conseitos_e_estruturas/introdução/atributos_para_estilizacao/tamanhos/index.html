Tamanhos

O tamanho dos elementos de um arquivo html podem ser definidos e manipulados via CSS, podendo utilisar como medida os seguintes parâmetros

  • px: tamanho absoluto, definido pela quantidade de pixels
  • %: define o tamanho como uma porcentagem relativa, normalmente, ao tamanho da página do site
  • em: define o tamanho como um porcentagem relativa, normalmente, ao tamanho do conteiner no qual o elemento se encontra
