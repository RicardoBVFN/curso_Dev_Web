Planos de fundo com CSS

Podemos tornar tanto cores como imagens em planos de fundo de nosso site, por meio do atributo "background" seguido de "image" para uma imagem ou "color" para uma cor.

Há ainda uma série de parâmetros de modificação para o plano de vundo, sendo os mais utilizados:

  • background-size: definir o tamanho do plano de fundo dentro de seu conteiner.
  • background-repeat: define como uma imagem de fundo irá se repetir dentro de seu conteiner, caso se deseje que esta se repita.
  • background-position: define a posição do plano de fundo dentro de seu conteiner.
  • background-attachment: define o comportamento do plano de fundo em relação ao scroll, se permanecera fixo ou não.
